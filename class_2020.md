HTML5 class, 2020, 1st semester.

'#' : 과제 미제출 (-1)

'@' : 과제 보완 (-1 또는 보완 시 감점 없음, 보완이 안되면 '#')

'^' : 과제 코드 일부 오류 (-0.5)

'!' : 노력과 아이디어 인정!
